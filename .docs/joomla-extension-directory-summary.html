Bears AI Chatbot for Joomla 5

Extension Directory Summary

Bears AI Chatbot transforms your Joomla 5 website into an intelligent support system using Retrieval-Augmented Generation (RAG) technology powered by IONOS AI Model Hub. The extension automatically ingests your Joomla articles, optional Kunena forum posts, and external URLs to create a searchable knowledge base that provides instant, accurate responses to user questions. With support for multiple AI models (Llama 3, Mixtral, Mistral), full WCAG 2.1 AA accessibility compliance, and a customizable chat interface featuring dark mode and responsive design, it delivers a professional AI-powered support experience that reduces support workload while maintaining strict content accuracy.

The comprehensive administrator component provides powerful analytics including token usage monitoring, cost tracking, keyword extraction with trending analysis, and detailed usage reporting with CSV export. Automated background processing through Joomla Scheduler ensures content stays synchronized, while the modular architecture includes a site module for the chat interface, task plugins for content processing, and a content plugin for real-time updates. Requirements: Joomla 5.x, PHP 8.1+, and an active IONOS AI Model Hub subscription with API credentials (visit cloud.ionos.com/managed/ai-model-hub to get started).
