Bears AI Chatbot for Joomla 5

Extension Directory Summary

Bears AI Chatbot for Joomla 5 is a comprehensive AI-powered knowledge base solution that transforms your Joomla website into an intelligent support system. Built with Retrieval-Augmented Generation (RAG) technology and powered by IONOS AI Model Hub, this extension automatically ingests your content to create a searchable knowledge base. The system features:

  • Automatic content ingestion from Joomla articles
  • Optional Kunena forum integration for forum posts
  • External URL support for additional knowledge sources
  • Customizable chat interface with flexible positioning options
  • Dark mode support and responsive design for all devices
  • Multiple AI model support including Llama 3, Mixtral, and Mistral
  • Full accessibility support with WCAG 2.1 AA compliance for inclusive user experience
  • Strict dataset-only answering ensuring accurate, relevant responses based solely on your content

The extension includes a powerful administrator component with comprehensive analytics, cost tracking, and keyword extraction capabilities. Site administrators can monitor and analyze their AI chatbot performance through:

Analytics & Management Features:

  • Token usage monitoring and spending trend analysis
  • User interaction tracking with detailed usage reports
  • Intelligent keyword analysis with configurable stop words
  • Multi-language support for keyword extraction
  • Automated document collection management through background processing
  • Joomla Scheduler integration for content synchronization
  • Collection rebuilding tools for maintenance
  • Trending keyword analysis to identify popular topics
  • Detailed usage reporting with CSV export functionality

Accessibility Features (WCAG 2.1 AA Compliant):

  • Screen reader compatibility with complete ARIA labeling and live regions
  • Full keyboard navigation with shortcuts (Ctrl+/ to open, Escape to close)
  • High contrast mode support and enhanced focus indicators
  • Reduced motion support for users with vestibular disorders
  • Multi-language accessibility with 15+ translatable screen reader announcements
  • Semantic HTML structure with proper roles and form labeling
  • Real-time status announcements for AI processing and loading states
  • Responsive design that works at all zoom levels up to 200%

Bears AI Chatbot provides everything needed to deploy and manage a professional AI-powered support system that:

  • Reduces support workload by automatically answering common questions
  • Improves user experience with instant, accurate responses
  • Maintains content accuracy through strict dataset-only responses
  • Scales efficiently with automated background processing
  • Provides actionable insights through comprehensive analytics
